Welcome To Pathfinder

Pathfinder seeks to find the shortest path between two points. This application visualizes various pathfinding algorithms in action, and more!

- Algorithm Visualization : Pressing on Dijkstra will visualizes dijkstra's algorithm trying to find the shortest path.

- Adding Walls : Click and Slide on the Grid to add a Wall.

- Generating Mazes : Click on Basic Maze or Walls Maze to generate mazes.

- Dragging Nodes : Click and Drag the Start or Finsih nodes to move them.
